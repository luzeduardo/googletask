## Locally
```bash
git clone git@github.com:luzeduardo/googletask
cd googletask
yarn install
yarn run build
```
Access: http://localhost:3000/

## Static html dist
```bash
git clone git@github.com:luzeduardo/googletask
cd googletask
yarn install
yarn run dev
```
Open the index.html inside dist folder
